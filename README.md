Виселица — игра на Python

Консольный проект с графическим интерфейсом, реализующий классическую игру «Виселица» на русском языке.

  Компьютер загадывает случайное слово из списка (длиной 4–8 букв).
  Игрок угадывает буквы, нажимая клавиши на клавиатуре или наблюдая за алфавитом справа.
  На отгадывание даётся 5 попыток.
  Неправильные буквы зачёркиваются в алфавите, угаданные — открываются в слове.
  Игра завершается победой (слово угадано) или поражением (закончились попытки).
  Кнопка «Заново» позволяет начать новую игру без перезапуска программы.

  
Использует встроенные модули tkinter для GUI и random для выбора слова.
